// Wayto //

Application specialisé dans les recommandations touristiques urbaines

MVP :

- Acces : Localisation
- ( Login )
- First Step : QUIZZ de personnalité
                    - Age
                    - Centres d'interets (  Vie de nuit / culturelle / gastronomie )
                    - Situation ( Solo / En couple / en Famille )
                    - Budget ( Petit / Moyen / Grand )

- Second step : transformer des données (femme, 21, solo, moyen) en un itinéraire (une tableau de coordonnées) grâce à un modèle IA (API)
- Home interface : Map
- Front : Fonction de recherche / *** Historique de navigation *** / *** Favoris ***
- Backlog : Calcul itinéraire , affichage sur map
