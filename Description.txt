// Wayto //

Application specialisé dans les recommandations touristiques urbaines

### Minimum Viable Product ###

- Acces : Localisation
- ( Login )
- Step 1 : Questionnaire de personnalité
                    - Age
                    - Centres d'interets (  Vie de nuit / culturelle / gastronomie )
                    - Situation ( Solo / En couple / en Famille )
                    - Budget ( Petit / Moyen / Grand )

- Step 2 : Transformer des données (femme, 21, solo, moyen) en DEUX itinéraire (deux tableaux de coordonnées) grâce à un modèle IA (API)
>>> Où récupère-t-on les données des lieux ? On les stocke en dur quelque part et on fait des requêtes SQL, ou on interroge l'API de Google Maps (ou quelques chose du genre)
>>> Quel modèle de machine learning utilise-t-on ? Comment l'entraine-t-on ?

- Step 3 : Choix de l'itinéraire grâce à la liste des lieux (bouton DEPART pour commencer l'itinéraire)

- Step 3 : Afficher l'itinéraire sur une carte (Google Directions API et Google Maps Android API)
